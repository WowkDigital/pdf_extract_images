# PDF Image & Text Extractor

A fast, secure, and entirely client-side web application to extract images and text from PDF files. No server upload required – all processing happens locally in your browser.

## 🚀 Live Demo

[Live Demo](https://wowkdigital.github.io/pdf_extract_images/)

## ✨ Features

- **Multi-File Support**: Drag and drop multiple PDF files at once.
- **Image Extraction**: Automatically identifies and extracts embedded images (XObjects) from PDF pages.
- **Text Extraction**: Recovers text while preserving as much structure and line breaks as possible.
- **Download as ZIP**: Download all extracted content (images and text) in a structured ZIP archive.
- **Privacy First**: Files never leave your computer. Processing is done 100% locally using JavaScript.
- **Modern UI**: Sleek dark mode design with glassmorphism effects and responsive layout.

## 🚀 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/pdf-image-extractor.git
   ```
2. **Open `index.html`**: Simply open the `index.html` file in any modern web browser (Chrome, Edge, Firefox, Safari).
3. **Upload Files**: Drag and drop your PDFs into the upload zone.
4. **Extract**: View your extracted images and text instantly, and download them individually or as a ZIP.

## 🛠️ Technology Stack

- **HTML5 & CSS3**: Semantic structure and modern styling (Vanilla CSS).
- **JavaScript (Vanilla)**: Core logic without heavy frameworks.
- **[PDF.js](https://mozilla.github.io/pdf.js/)**: Robust library for reading and parsing PDF files.
- **[JSZip](https://stuk.github.io/jszip/)**: Library for creating and downloading ZIP archives.
